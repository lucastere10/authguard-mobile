# AuthGuard Mobile App

A modern mobile authentication demo built with React Native that allows users to sign in via social login and authenticate web sessions by scanning QR codes. This mobile app utilizes Gluestack UI for a consistent and elegant design and integrates a QR code scanner to send mobile login confirmations to the web application.

---

## 🚀 Features

- **Social Login**  
  Easily sign in using popular social platforms, providing a smooth and secure login experience.

- **QR Code Scanner**  
  Authenticate web sessions by scanning a QR code. Once scanned, the mobile app sends the login confirmation to the backend for seamless authentication on the web.

- **Modern UI Components**  
  Built with Gluestack UI for a refined and user-friendly interface.

## 🔄 Authentication Flow

1. **Social Login**  
   - The user initiates login using their preferred social provider.
   - Upon successful authentication, the app receives an access token for further interactions.

2. **QR Code Authentication for Web**  
   - The web app displays a QR code generated by the backend.
   - The user scans the QR code using the mobile app.
   - The app extracts the session identifier and sends a confirmation to the backend.
   - The backend verifies the session and logs the user into the web application.

---

## 📦 Tech Stack

- **Framework:** React Native  
- **UI Library:** Gluestack UI  
- **Authentication:** Social Login (via supported providers)  
- **QR Code:** Integrated QR code scanner for mobile-to-web authentication  
- **Assets:** Custom images located in `public/images` (home.png, login.png)

## Getting Started

### Installation

Clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd authguard-mobile-app
npm install
# or
yarn install
```

### Running the App

Run the app on your emulator or connected device:

```bash
npx react-native run-android
# or
npx react-native run-ios
```
